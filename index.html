How to Play

← → / A DRun
↑ / W / SpaceJump (tap = short hop)
E / EnterOpen a math gate
NNotebook — the math behind every gate
RRestart level
EscClose panel · level select

Pits and spikes respawn you instantly — the only real obstacle is the math gate in each level. Open it, set your answer, commit to a prediction (your landing spot, your arrival time, the tangent slope…), and Run it: the physics grades the answer, the math grades the prediction. Gold = first try + correct prediction. A lucky run with a wrong prediction caps at silver — the game can tell the difference between knowing and wiggling.
